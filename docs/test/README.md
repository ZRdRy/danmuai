# test — 实验性内置人格登记

本目录用于登记**非默认激活**、便于试验与后续撤回的内置人格（如「测试」）。源码仍以 `app/personae.py` 为准；此处保存完整提示词副本与增删改清单，避免只在代码里翻找。

| 文件 | 说明 |
|------|------|
| [persona-测试.md](persona-测试.md) | 内置人格「测试」：提示词全文、代码落点、修改/删除步骤 |

## 何时更新

- 在 `BUILTIN_PERSONAE` 中新增/修改/删除试验用人格时，同步更新对应 `persona-*.md`。
- 从代码中彻底移除某人格后，可将对应文档移至 `docs/archive/` 或删除本目录下该文件，并更新本 README 表格。

## 与产品人格的关系

- 人格工坊列表来自 `PersonaManager.list()`（内置 + 用户自定义）。
- 「测试」**未**写入 `PersonaManager.DEFAULT_ACTIVE`；需在 Web 人格工坊手动勾选后才会参与随机轮询。
