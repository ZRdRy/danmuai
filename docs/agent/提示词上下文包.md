## 项目一句话说明

DanmuAI 是 **Windows 桌面 AI 弹幕助手**：按固定间隔截取屏幕，调用视觉语言模型生成短弹幕，在 **Qt 透明置顶 Overlay** 上展示；配置与启停通过本地 **Web 控制台**（FastAPI + pywebview）完成。
